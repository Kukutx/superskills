# Prompt Template

Use this template when running any skill in `superskills`.

## Standard version

```text
Use the [skill name] skill.

Goal:
Context:
Input material:
Audience:
Constraints:
Preferred style:
Output format:
Must include:
Must avoid:
```

## Minimal version

```text
Use the [skill name] skill.

Goal:
Constraints:
Output:
```

## Advanced version

```text
Use the [skill name] skill.

Task:
Audience:
Background:
Available assets:
Current problem:
Constraints:
Tone/style:
Format:
Quality bar:
Examples I like:
Examples I dislike:
Decision needed:
Deadline or priority:
```

## Conversion version

Use this when turning a rough idea into a polished request.

```text
Use the Prompt Optimizer skill.

Rough idea:
Target AI tool:
Desired output:
Context:
Constraints:
Examples:
```

## Skill creation version

Use this when creating a new skill.

```text
Use the Skill Builder skill.

Skill domain:
Main task:
Typical input:
Desired output:
Common mistakes to avoid:
Default assumptions:
Quality bar:
```

## Review version

Use this when improving an existing skill.

```text
Use the Skill Review skill.

Current skill:
Problems noticed:
Desired behavior:
Examples of bad output:
Examples of good output:
```
