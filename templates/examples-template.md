## Edge cases

| Case | Expected behavior |
| --- | --- |
| Missing context | Make reasonable assumptions and continue |
| Conflicting constraints | Point out conflict and choose the safest assumption |
| Too broad request | Provide a structured first pass instead of asking many questions |
| User asks for only final output | Remove explanations and provide final output only |
