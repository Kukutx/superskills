# Skill Builder Examples

## Example 1: Create a support email skill

### Input

```text
帮我把这个重复工作流变成 superskills 里的一个 skill：
我经常需要回复客户邮件，要专业、简短、不要乱承诺，要保留谈判空间。
```

### Expected output pattern

```text
Recommended skill path:
skills/writing/customer-support-email/

Maturity target:
usable → refined

skill.md draft:
...

prompt-template.md draft:
...
```

## Bad output

```text
你可以创建一个叫 email 的 skill，里面写“帮我写邮件”。
```

Why bad:

- Name too broad.
- No constraints.
- No examples.
- No quality bar.
