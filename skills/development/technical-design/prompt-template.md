# Technical Design Prompt Template

## Standard request

```text
Use the Technical Design Skill.

Goal:
Current stack:
Requirements:
Existing system:
Constraints:
Known risks:
Output depth: brief / normal / deep
```

## Feature design request

```text
Use the Technical Design Skill.

Feature:
Users:
User flow:
Data needed:
API needed:
Permissions:
Current stack:
Constraints:
Output: approach, architecture, data model, API contract, steps, risks, tests
```

## Minimal request

```text
帮我设计这个功能的技术方案，要求简单可落地：

[feature description]
```
