# Shopify Development Prompt Template

```text
Use the Shopify Development Skill.

Store type:
Theme:
Goal:
Current issue:
Code/context:
Constraints:
Output depth: brief / normal / deep
```

```text
Use the Shopify Development Skill.

Theme:
Page/template:
Desired change:
Current code:
No-app requirement: yes / no
Output: implementation steps, code, placement, risks, validation
```
