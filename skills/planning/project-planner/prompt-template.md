# Project Planner Prompt Template

## Standard request

```text
Use the Project Planner Skill.

Goal:
Current status:
Deadline:
Resources:
Constraints:
Known risks:
Preferred output: roadmap / checklist / sprint plan / launch plan
```

## Minimal request

```text
帮我把这个目标拆成可执行计划：

[goal]
```

## Software project request

```text
Use the Project Planner Skill.

Project:
Current code/product status:
Target users:
Required features:
Tech stack:
Deadline:
Constraints:
Output: phases, tasks, dependencies, risks, first 3 actions
```
