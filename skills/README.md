# Skills Index

This folder contains reusable AI skills grouped by domain.

## Categories

| Category | Description |
| --- | --- |
| `creative` | Physical products, engraving, illustration, image generation, gift design |
| `development` | Code review, architecture, debugging, implementation planning |
| `ecommerce` | Shopify, product pages, conversion optimization, store operations |
| `design` | App store assets, feature graphics, brand visuals, landing pages |
| `writing` | Business emails, concise copywriting, professional communication |

## Naming rules

- Use lowercase folder names.
- Use hyphenated names: `xtool-f1-engraving`, not `xtoolF1Engraving`.
- Prefer domain + task names.
- Avoid overly narrow names unless the skill is tool-specific.

## Skill maturity levels

| Level | Meaning |
| --- | --- |
| `draft` | Rough idea, not stable yet |
| `usable` | Can be used in real tasks |
| `refined` | Has examples and constraints |
| `stable` | Mature, rarely changes |

## Recommended files per skill

```txt
skill.md
prompt-template.md
examples.md
changelog.md
```
