# Research Brief Prompt Template

## Standard request

```text
Use the Research Brief Skill.

Research question:
Decision context:
Sources I already have:
Need latest info: yes / no
Output depth: brief / normal / deep
Must compare:
Must include:
```

## Minimal request

```text
帮我研究这个问题并给出简洁决策建议：

[research question]
```

## Competitor research request

```text
Use the Research Brief Skill.

Topic:
Competitors:
Market:
Decision I need to make:
Need latest info: yes
Output: summary, comparison table, risks, recommendation
```
