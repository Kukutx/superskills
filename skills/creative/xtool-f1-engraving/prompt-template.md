# xTool F1 Engraving Prompt Template

## Standard request

```text
Use the xTool F1 Engraving Designer Skill.

用途：
材料：
尺寸：
风格：
是否需要文字：
文字内容：
是否需要切割外形：
目标受众：
我想要的感觉：
输出：请给我图案结构、雕刻层级、图像生成提示词、SVG 制作建议。
```

## Minimal request

```text
按 xTool F1 雕刻设计师模式，给我一个适合 [材料] 的 [用途] 图案。
风格：[风格]
文字：[文字或无]
尺寸：[尺寸]
```

## Image generation prompt

```text
Create a black and white vector-style laser engraving design for xTool F1.
Canvas size: 100 mm × 100 mm.
High contrast, clean outlines, bold shapes, engraving-friendly, no gradients, no shadows, no realistic texture, no tiny details.
Style: [style]
Subject: [subject]
Text: [text]
Use clear negative space, balanced composition, suitable for SVG conversion and laser engraving.
```

## Chinese image generation prompt

```text
为 xTool F1 设计一个黑白矢量风激光雕刻图案。
画布尺寸：100 mm × 100 mm。
要求：高对比、粗线条、干净轮廓、适合雕刻、不要渐变、不要阴影、不要照片质感、不要过密细节。
风格：[填写风格]
主题：[填写主题]
文字：[填写文字]
构图要有留白，适合转 SVG 并导入 xTool Creative Space。
```
