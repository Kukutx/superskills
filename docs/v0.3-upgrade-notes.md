# v0.3 Upgrade Notes

`v0.3` upgrades `superskills` toward a more mature, reusable, Europe/US-market-ready personal AI skills system.

## Added docs

- `docs/personal-defaults.md`
- `docs/western-market-guidelines.md`
- `docs/workflow-recipes.md`

## Added skills

| Skill | Maturity | Purpose |
| --- | --- | --- |
| `meta/skill-router` | refined | Choose the right skill or workflow for a task |
| `product/prd-builder` | refined | Turn ideas into PRDs, MVP scope, user stories, and acceptance criteria |
| `development/implementation-plan` | refined | Convert a technical design into concrete file/task-level implementation steps |
| `design/image-review-refiner` | refined | Review generated images and create the next refinement prompt |
| `marketing/product-positioning` | refined | Define product positioning, audience, benefits, and messaging |
| `writing/app-store-copy` | refined | Create App Store / Google Play copy |
| `operations/release-checklist` | refined | Prepare app/web/Shopify releases with a practical launch checklist |
| `creative/ip-safe-creative-adapter` | refined | Convert copyrighted/trademarked inspirations into original, safer creative directions |

## Why this upgrade matters

The repository already had useful individual skills. This upgrade makes it more like a personal AI operating system:

1. `personal-defaults.md` defines how AI should behave for the user by default.
2. `western-market-guidelines.md` prevents outputs from feeling too local, generic, or AI-generated.
3. `skill-router` helps choose the right skill when the task is vague.
4. `workflow-recipes.md` combines multiple skills into repeatable workflows.
5. Product, design, development, marketing, writing, and release skills cover most solo-founder / full-stack-builder scenarios.
