# Workflow Recipes

This document combines individual skills into repeatable workflows.

## Choose a skill

```txt
meta/skill-router
```

Use this first when the task is vague or spans multiple domains.

## New product or feature

```txt
product/prd-builder
→ development/technical-design
→ development/implementation-plan
→ development/code-review
→ operations/release-checklist
```

Use when turning an app or SaaS idea into buildable work.

## Debug and fix

```txt
development/bug-diagnosis
→ development/implementation-plan
→ development/code-review
```

Use when a feature is broken or logs are available.

## App Store / Google Play launch

```txt
marketing/product-positioning
→ writing/app-store-copy
→ design/app-store-assets
→ design/image-review-refiner
→ operations/release-checklist
```

Use when preparing store listing, screenshots, feature graphic, and release copy.

## Image generation workflow

```txt
creative/image-prompt-director
→ design/image-review-refiner
→ creative/image-prompt-director
```

Use this loop until the generated image is production-ready.

## xTool F1 engraving workflow

```txt
creative/ip-safe-creative-adapter
→ creative/xtool-f1-engraving
→ creative/image-prompt-director
→ design/image-review-refiner
```

Use when creating laser engraving artwork, especially if the inspiration comes from a known IP.

## Shopify workflow

```txt
research/research-brief
→ ecommerce/shopify-dev
→ marketing/product-positioning
→ writing/business-email
→ operations/sop-builder
```

Use for store setup, product page optimization, supplier communication, and repeated operations.

## Repeated task into reusable system

```txt
meta/prompt-optimizer
→ meta/skill-builder
→ operations/sop-builder
```

Use when a prompt or manual workflow appears more than 3 times.
