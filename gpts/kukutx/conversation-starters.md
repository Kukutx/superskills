# Conversation Starters

Use these as starter prompts for the Private GPT.

```text
先帮我选 skill，再执行：我要做一个欧美市场的 App 素材。
```

```text
用 superskills 处理这个任务：帮我把这个产品想法变成 PRD，并拆成开发任务。
```

```text
用 xTool F1 workflow 帮我设计一个原创、安全、适合雕刻的礼物图案。
```

```text
帮我 review 这张生成图，指出问题并给我下一轮优化 prompt。
```

```text
帮我把这个功能做技术设计和 implementation plan。
```

```text
帮我写一个欧美市场友好的 App Store 描述和截图文案。
```

```text
帮我 review 这段代码，优先找真实 bug、安全风险和上线风险。
```

```text
把这个重复流程整理成 SOP，方便我以后复用。
```
